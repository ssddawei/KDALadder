关于 KDALadder.top

app 主要功能

  1. 记分：记录赛局比分
  2. 看板：多个终端实时同步比分
  3. 排名：根据赛局kda得分计算排名

kda 记录规则

  • Kill（杀分）：击球后，直接落入对方场内，计为杀分。杀球，平抽挡等快球导致对方接球失误的，计为杀分
  • Death（主动失误）：击球后，出界或落网失分的，计为主动失误。因对手杀球等高质量球的接球失误不算作主动失误（计为对方杀分）
  • Assist（助杀）：队友杀分后，自动获取一个助杀，用于评价配合程度
  • KDA 得分：kda = ( kill + assist/4 ) / death * 0.7 + ( win / (win+loss) ) * 0.3 说明：kda 得分主要反映杀分比率，同时加上胜率，分别占据7/3的权重，大于1代表得分多于失误，少于1代表失误多于得分

kda 团体天梯

  • 根据团体内成员 kda 得分排行

kda 全服天梯

  • 全服天梯的个人 kda 得分会乘以团体加成修正，再进行排行
  • 团体加成：根据团体内成员的胜率计算加权标准差（场次为权值），加权标准差作为团体加成（0%-100%）。团体成员水平越接近，加成越高，反之越低。因为水平差距越大，则 kda 成绩越不准确，只能取下限值参与排行
