# KDALadder 记分器
## 主要功能
- 计分
- 多屏同步
- KDA 天梯排行
## 概述
记录合作类竞技项目(羽毛球双打)的 KDA(kill,death,assist) 表现，生成一个天梯排名，提高娱乐性
